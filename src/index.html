✓Import into Premiere

  1. Open (or create) your Premiere Pro project.
  2. Go to File → Import… and select the exported .xml file.
  3. Premiere will build a new sequence named Storyboard Animatic with every frame already trimmed to the duration you recorded.
  4. The clips link to the images in the folder you loaded — keep that folder in place, or use Link Media if you move it.
